Antigravity: MCP servers + skills

Connect tools carefully. Add skills only from sources you trust.

Install an MCP server

  • Open Antigravity's Agent Panel.
  • Use the three-dot menu and open MCP Servers / Manage MCP Servers.
  • Use the store when available, or open the raw MCP config.
  • Add the server config from the server's README.
  • Restart, reload, or toggle the server, then test one tool.
Ask your agent:

Help me add this MCP server to Antigravity.
Read the README, explain what access it needs,
edit the MCP config, and verify the tools appear.

Add a skill

  • A skill is reusable context: instructions, examples, scripts, or workflow memory.
  • Find the skill repo or marketplace page.
  • Read its README before installing.
  • Install or import it using the tool's instructions.
  • Restart/reload the agent and ask it to summarize the skill.
Ask your agent:

Install this skill for my project.
Check the repo first, explain what it changes,
then confirm how I should invoke it.

Rule of thumb: MCP servers can touch external services and local files. Skills can shape how the agent behaves. Treat both like plugins, not harmless text.
