Node, npm, npx

One install gives you all three.

Route A: ask your agent

Best if you're unsure. Tell Claude Code / Cursor / Antigravity to inspect your machine, install what is missing, and verify the result.

Check whether Node, npm, and npx are installed on my machine.

If anything is missing, guide me through installing Node LTS for my operating system.

After installation, verify:
node -v
npm -v
npx -v

Do not assume it worked until all three commands print versions.

Route B: copy commands

Use the OS-specific slides next. Paste commands into PowerShell on Windows or Terminal on Mac.

  • Install Node LTS, not random packages.
  • Close and reopen the terminal after install.
  • Verify before starting workshop commands.

Windows route — PowerShell

Install Node LTS on Windows

Ask the agent

I am on Windows.

Help me install Node.js LTS using PowerShell.
Check whether winget is available.
If winget works, use it.
If it does not, guide me to the official Node.js LTS installer.

After install, make me close and reopen PowerShell, then verify node, npm, and npx.
powershell
winget install OpenJS.NodeJS.LTS

# Close PowerShell. Open a new PowerShell window.
node -v
npm -v
npx -v

If `winget` is not available, download the LTS installer from nodejs.org and run the same three checks after installing.

Mac route — Terminal

Install Homebrew, then Node

Ask the agent

I am on macOS.

Help me check whether Homebrew is installed.
If Homebrew is missing, guide me through installing it from brew.sh.
After Homebrew works, install Node using brew.

If Homebrew prints post-install PATH instructions, help me apply them.
Then verify node, npm, and npx.
terminal
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Follow any PATH instructions Homebrew prints.
brew install node

node -v
npm -v
npx -v

If `brew` is not found after install, close Terminal and reopen it, then follow Homebrew's printed PATH instructions.

The rule for workshop setup

Let the agent do the boring parts

Your job is to verify, not memorize install commands.

Good agent prompt

Set up my machine for this frontend workshop.

I need Node, npm, and npx working.
Ask me before installing anything.
Explain each step briefly.
Verify with version commands at the end.

Keep nearby

  • Windows: PowerShell + winget or Node LTS installer.
  • Mac: Terminal + Homebrew + brew install node.
  • Always verify with node, npm, and npx.
  • If stuck, paste the exact error into your agent.
