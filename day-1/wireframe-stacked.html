Normal Flow

Before flexbox and grid, everything was normal flow: block elements stacking top-to-bottom, inline elements flowing left-to-right like text.

Normal flow is still the right tool for many layouts — especially simple stacked content like this page.

How normal flow works

→ Block elements (div, p, h1, section...) take the full width available and stack vertically. No extra CSS needed.
→ Inline elements (span, a, strong...) flow like text — left to right, wrapping when they run out of space.
→ You can space stacked blocks with margin-top and margin-bottom. No flexbox required.
→ Normal flow is responsive by default — stack layouts work fine on mobile without any media queries.
→ Most of what you see on the web is normal flow + small amounts of flexbox/grid for the parts that need two-dimensional control.
