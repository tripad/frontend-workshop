CSS Grid

Grid arranges items in rows and columns simultaneously. You define the structure; the browser places items.

Change the number of columns and watch items reflow.

.grid-container { grid-template-columns: repeat(3, 1fr); }

What these properties do

display: grid Activates grid layout. Children become grid items.
grid-template-columns Defines the column structure. 1fr = one equal fraction of available space. 200px 1fr = fixed sidebar + flexible content.
gap Space between both rows and columns. You can also use row-gap and column-gap separately.
grid-template-rows Same idea, but for rows. Often left to auto (browser decides row heights).

Grid vs. Flexbox — when to use which

Flexbox: one-dimensional layouts — a nav bar, a row of cards, a centered element.

Grid: two-dimensional layouts — a full page layout, an image gallery, a dashboard with rows and columns that need to line up.

In practice you'll use both. They work fine together.
