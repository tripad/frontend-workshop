HTML = the skeleton.

It's just labeled boxes, nested inside other boxes.

<header>
  <h1>Hello</h1>
</header>
<main>
  <p>This is a page.</p>

</main>

<header>, <main>, <button>, <p>, <img> — these are called elements or tags.

API = the menu between frontend and backend.

Frontend

That handshake, standardized, is an API.
